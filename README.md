# PeakWell Company Analytics Portfolio

## Project Overview
This project demonstrates SQL and analytics skills using a **realistic dataset representing PeakWell company sales**.  
The goal is to provide **executive insights** into the company’s performance.

## Dataset
- **Tables:** 5 normalized tables — `locations`, `customers`, `products`, `orders`, `order_items`  
- **Customers:** 50  
- **Products:** 20 across 5 categories  
- **Orders:** 100  
- **Order Items:** 120  
- **Locations:** 5 US cities  
- **Features:** realistic revenue, product categories, dates, and churn flags  

## Executive Dashboard KPIs
This dataset supports telling the company story through key metrics:  

1. **Total Revenue** – overall company revenue from all orders  
2. **Total Orders / Customers** – order volume and unique customers  
3. **Churn Rate / Retention** – % of one-time customers vs returning  
4. **Top Product Categories** – highest-performing product categories by revenue or orders  
5. **Revenue by Location** – performance by city for business insights  

## Skills Demonstrated
- SQL for **joins, aggregations, filtering, and grouping**  
- Designing **normalized relational tables**  
- Building **dashboard-ready datasets**  
- Translating raw sales data into **executive insights**
